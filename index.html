How to get .tgs files?

Telegram stickers are protected. Here is the easiest way to download them.

1

Use Telegram Desktop (PC/Mac)

  • Open Telegram Desktop app.
  • Find the sticker you want in a chat.
  • Right Click on the sticker.
  • Select "Save Sticker As...".
  • It will download as a .tgs file.
2

Use a Download Bot (Mobile/PC)

If you are on mobile or can't save directly:

  • Search for @StickerDownloadBot on Telegram.
  • Send the sticker you want to the bot.
  • The bot will reply with the .tgs file.
  • Download that file and upload it here.

Pro Tip: You usually cannot drag a sticker directly from the Telegram chat window into the browser. You must save it as a file to your computer first!
